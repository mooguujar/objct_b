理享家（金融理财类 App，面向年轻用户的智能财富管理与投资决策助手）
一、产品定位与核心理念

品牌定位：
「理享家」是一个懂年轻人的智能财富助理，帮助用户轻松了解资产分布、制定理财目标、自动规划投资组合，并通过个性化推荐实现财富增长。

目标用户：
20–35 岁城市年轻群体（白领 / 自由职业者 / 新手投资者）

核心价值主张：

“让理财像刷短视频一样轻松，让财富增长变成日常的一部分。”

📱 二、App 总体架构
一级导航 功能定位
首页（Dashboard） 资产总览、目标达成、智能建议
理财（Invest） 产品推荐、组合规划、风险评估
洞察（Insight） 消费分析、理财报告、情绪分析
社区（Community） 理财经验分享、财经话题互动
我的（Profile） 用户资产管理、设置与成长体系
🧭 三、页面结构与功能说明
1️⃣ 首页（Dashboard）

核心目标： 一眼看懂自己的财富状态与目标进展

页面布局：

顶部区域 — 资产概览卡

动态净资产数值（实时刷新动画）

收益曲线（周/月切换）

「AI 智能总结」：一句话描述理财状态（如“稳健上涨，建议保持持仓”）

中部区域 — 目标追踪卡片组

每个目标（如“旅行基金”、“买房首付”）都有独立小卡，展示当前完成率、时间进度、投入金额

点击进入目标详情页

底部区域 — 推荐与提示

“为你精选”模块：基于风险偏好与现金流情况推荐理财产品（货基、定投基金、储蓄计划）

「智能提醒」：如“你的应急金不足 2 个月生活费”“本周消费过高”等

交互与细节：

下拉刷新资产数据（带流光动画）

资产曲线触摸可查看具体收益点数据

智能建议气泡在数据变化后自动弹出

2️⃣ 理财页（Invest）

核心目标： 简化理财产品选择过程，降低决策焦虑

页面布局：

顶部区域 — 投资总览

当前持仓金额、平均年化、风险等级标签

一键查看「我的投资组合」

智能组合推荐区

根据风险等级推荐 3 种方案：
「稳健保值」/「均衡成长」/「进取进阶」

卡片展示：预期收益区间 + 投资比例图 + 推荐说明

点击后进入组合详情页（含历史回测）

基金精选 / 定投计划

推荐基金（含主题标签：新能源、消费、AI 等）

定投助手：设置目标金额 → 自动生成定投策略

底部固定浮层 — 快捷入口

「创建新组合」

「AI 理财问答」：语音输入问题（如“我适合买债券基金吗？”）

3️⃣ 洞察页（Insight）

核心目标： 让理财“可视化 + 有故事”，强化用户自我洞察

页面布局：

顶部区域 — 财富健康仪表盘

综合得分（理财活跃度、收益率、风险控制）

动态环形进度条 + 表情反馈（AI 语气：“你的财务健康度正在上升～”）

中部 — 消费与投资洞察

「消费分布」饼图（按类别：餐饮/娱乐/房租等）

「投资构成」环图（基金/存款/保险/现金）

可滑动切换视图周期（月/季度）

底部 — AI 洞察卡片流

类似“理财日签”：

「你上月的支出下降了 12%，节奏掌握得很好！」

「本月基金收益主要来自新能源主题，是否考虑定投？」

卡片可左右滑动保存或忽略

4️⃣ 社区页（Community）

核心目标： 打造轻互动式理财交流场景

页面布局：

顶部标签栏

热门 / 新手问答 / 实盘分享 / 投资心态

主内容区

内容卡流（每条动态含用户头像、理财心得、标签、点赞评论数）

视频/图文混排，带理财话题标签

底部功能

「发布」按钮悬浮在右下角：支持文字/语音/截图上传

支持“匿名分享”，昵称自动生成（如“理智的基金猫”）

5️⃣ 我的页（Profile）

核心目标： 用户资产、账户与成长体系中心

页面布局：

顶部 — 用户信息与资产汇总

头像 + 用户等级（如“理财新手 Lv.2”）

总资产 + 净收益 + 绑定账户数

背景渐层动画随收益正负变色

中部 — 我的服务区

「我的投资组合」

「风险测评」

「理财目标」

「自动投资计划」

底部 — 系统与设置

夜间模式切换、账户绑定、安全设置、隐私中心

「关于理享家」卡片带品牌理念简介

⚙️ 四、交互与设计思路
场景 动效与反馈 风格特点
首页数据更新 金融流光动效 + 数字渐变跳动 明亮、流畅、科技感
收益曲线交互 触点高亮 + 轻振动反馈 可视化体验强化
AI 建议弹窗 弹出动画带半透明模糊背景 温和提示，不打断操作
投资切换动画 卡片滑动 + 渐隐渐显 类似 Apple Card 样式
夜间模式 深灰背景 + 品牌蓝绿渐层点缀 稳重但年轻

💬 五、用户旅程总结

理享家通过“轻操作 + 智能分析 + 情绪激励”
实现一个让年轻人理解金钱、管理目标、持续理财的智能助手体验。

理享家（FinSmart）UI Style Guide

Design Language: Rational · Calm · Minimal · Data-driven

一、品牌与视觉理念
项目 描述
品牌定位 年轻用户的智能财富管理与投资决策助手
视觉关键词 冷静 · 智能 · 轻盈 · 可信赖
设计哲学 信息应清晰可靠，界面应简洁克制；让数据自己“说话”。
视觉体验目标 极简、统一、节奏平衡。淡化装饰，突出理性层次与交互反馈。
二、色彩系统（Color System）
🎨 主色调体系
名称 HEX 用途
主色（Primary Blue） #1C5CEA 品牌主色、按钮、选中态、数据高亮
深蓝（Dark Navy） #0D1733 背景深层、底部导航栏
浅蓝灰（Cool Gray） #F4F6FA 页面背景、表单底层
中性灰（Neutral Gray） #A5ADBA 次级文字、图标线条

警示红（Alert Red） #E04F5F 风险提示、负收益（唯一暖色）
成功绿（Positive Green） #3BC371 收益上升、完成状态

💡 所有视觉对比基于深浅层次变化完成，不依赖暖色或复杂渐层。

名称 HEX 用途

三、字体系统（Typography）
层级 字体 字重 字号(px) 用途
H1 思源黑体 / Inter Bold 28 页面主标题
H2 思源黑体 / Inter SemiBold 22 模块标题
H3 思源黑体 / Inter Medium 18 子模块、小节标题
Body L 思源黑体 / Inter Regular 16 正文主要文字
Body M 思源黑体 / Inter Regular 14 辅助文本、表单项

排版特征：

所有标题保持左对齐，增强结构逻辑。

数字与货币符号分层，重要数字字体略粗。

段落最大宽度 ≈ 屏幕宽度 − 24dp。

四、布局与间距（Layout & Spacing）
元素 圆角 阴影 间距
卡片 Card 12dp 极轻阴影 0 2 8 rgba(0,0,0,0.04) 外边距 16dp
按钮 Button 10dp 无阴影 内边距 16–20dp
输入框 Input 8dp 底部线条高亮 上下内距 12dp
弹层 Modal/Sheet 顶部圆角 24dp 背景模糊 外边距 24dp
图表区块 无圆角 背景透明 上下留白 24dp

所有组件采用 冷灰层叠逻辑 区分信息层级，而非色块或阴影对比。

五、组件视觉规范（Key Components）
1️⃣ 按钮（Buttons）
类型 样式 交互反馈
主按钮 Primary 填充主色 #1C5CEA 白字 按下时透明度 80%，无明显放缩
次按钮 Secondary 线框主色 + 白底 悬浮时填充浅蓝灰
文字按钮 Text 无边框，文字主色 点击后下划线渐显
禁用状态 填充浅灰文字淡出 无动效反馈
2️⃣ 输入框（Input Fields）

背景：#F4F6FA

聚焦态：底线变主色 #1C5CEA

字体：14px

占位文字：#A5ADBA

提示信息使用浅灰 #B4B8C5

3️⃣ 数据卡片（Data Card）

结构：标题 + 主数据 + 辅助说明

背景 #FFFFFF，圆角 12dp，无渐层。

主数据字体 Inter SemiBold 20px，颜色根据状态：

正收益 #3BC371

负收益 #E04F5F

普通数据 #1C5CEA

点击反馈：浅灰高亮（#E8EBF2）无阴影。

4️⃣ 标签（Tag）
类型 背景色 字体颜色
普通标签 #E8EBF2 #1C5CEA
激活标签 #1C5CEA #FFFFFF
警示标签 #E04F5F #FFFFFF
状态徽章 #3BC371 #FFFFFF
5️⃣ 图表（Charts）

背景透明，无网格填充；强调线条数据。

折线颜色：#1C5CEA 主线，次线使用浅灰蓝 #A5ADBA。

柱状图使用半透明主色。

触点交互时，点亮高亮圈 + 动态数字浮层。

禁止使用渐变或装饰性发光。

6️⃣ 弹窗（Modal & Bottom Sheet）

背景：#FFFFFF，顶部圆角 24dp。

遮罩层：rgba(0,0,0,0.4)。

出场动效：自底向上滑入（200ms，ease-out）。

元素对齐：居中，操作区按钮成对布局。

7️⃣ 顶部导航（App Bar）

背景：#FFFFFF（浅模式）/ #0D1733（暗模式）。

标题字体：H3（18px），主色或白色。

图标：线性描边 #1C5CEA，选中时填充主色。

8️⃣ 底部导航栏（Tab Bar）
状态 图标 标签文字
未选中 线性灰 #A5ADBA 灰色文字
选中 填充主色 #1C5CEA 主色文字
背景 #FFFFFF（浅模式）/ #0D1733（暗模式） 轻分割线 #D4D8E2
六、动效系统（Motion System）
场景 动效表现 时长 曲线
页面切换 内容淡入 + 上浮 8dp 220ms ease-out
按钮点击 透明度 0.8 → 恢复 120ms linear
数据更新 数字轻跳动 + 渐变亮度闪烁 180ms ease-in-out
加载中状态 细线蓝环旋转（不发光） 1.2s 循环 linear
提示出现 自下滑入 + 透明度渐变 200ms ease-in-out

动效风格关键词：
克制、轻量、功能导向。
所有动画服务于信息呈现，不做炫技动效。

七、图标系统（Iconography）

风格：线性矢量，圆角端点，笔画宽度 2px。

颜色：主色蓝 #1C5CEA 或中性灰 #A5ADBA。

图标统一使用 24 × 24 dp 网格。

禁止内阴影、渐层、立体描边。

八、整体界面基调
属性 表达
视觉节奏 留白多、对齐严谨、颜色克制
配色气质 冷静中性、避免情绪化暖色
信息层级 颜色弱化，排版强化
整体感受 专业、可信、低干扰、智能感强

Apple Stocks / Revolut / Monzo Business （信息导向与留白结构）
