# PM2 启动问题排查步骤

## 1. 查看 PM2 日志

```bash
# 查看详细日志
pm2 logs nuxtjs-app --lines 50

# 查看错误日志
pm2 logs nuxtjs-app --err

# 查看实时日志
pm2 logs nuxtjs-app
```

## 2. 检查应用状态详情

```bash
# 查看详细信息
pm2 describe nuxtjs-app

# 查看进程信息
pm2 info nuxtjs-app
```

## 3. 手动测试启动

```bash
# 先停止 PM2 中的应用
pm2 stop nuxtjs-app
pm2 delete nuxtjs-app

# 手动测试是否可以运行
cd /home/objct_b/nuxtjs
node --experimental-fetch .output/server/index.mjs
```

## 4. 检查文件是否存在

```bash
# 检查必要文件
ls -la .output/server/index.mjs
ls -la ecosystem.config.cjs
ls -la .env
```

## 5. 检查环境变量

```bash
# 查看当前环境变量
cat .env

# 测试时手动设置环境变量
export NODE_ENV=production
export PORT=3000
node --experimental-fetch .output/server/index.mjs
```
