# 岛屿社交平台 - Web端

## 技术栈

- Nuxt.js 3
- Element Plus UI组件库
- Pinia 状态管理
- TypeScript
- TailwindCSS

## 项目结构

```
nuxtjs/
├── pages/          # 页面目录
├── components/     # 组件目录
├── composables/    # 组合式函数
├── utils/          # 工具函数
├── api/            # API接口
├── store/          # Pinia状态管理
├── layouts/        # 布局文件
└── assets/         # 静态资源
```

## 开发命令

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```
