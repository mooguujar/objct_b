# 岛屿社交平台 - UniApp端

## 技术栈

- Vue 3 + TypeScript
- uView Plus UI组件库
- Pinia 状态管理
- Axios 网络请求

## 项目结构

```
uniapp/
├── pages/          # 页面目录
├── components/     # 组件目录
├── utils/          # 工具函数
├── api/            # API接口
├── store/          # Pinia状态管理
└── static/         # 静态资源
```

## 开发命令

```bash
# 安装依赖
npm install

# H5开发
npm run dev:h5

# 微信小程序开发
npm run dev:mp-weixin
```
